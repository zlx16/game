# 像素冒险闯关（马里奥风）- 纯 HTML Canvas

这是一个**零依赖**的横版闯关小游戏：平台跳跃、收集金币、躲尖刺、踩怪，到旗帜通关。

## 运行方式（推荐）

很多浏览器对 `file://` 直接打开的脚本会有安全限制。建议用一个本地静态服务器启动。

### 方式 A：Python（最省事）

1. 在本目录打开 PowerShell
2. 运行：

```powershell
python -m http.server 5173
```

3. 用浏览器打开 `http://localhost:5173/`

### 方式 B：直接双击（可能可用）

直接双击打开 `index.html`。如果画面不显示或控制台报错，请改用“方式 A”。

## 操作

- **移动**：← / →
- **跳跃**：↑ 或 Space
- **冲刺**：Z
- **暂停**：Esc
- **重开**：R
- **过关**：碰到右侧的旗帜

## 文件结构

- `index.html`：入口页面
- `style.css`：页面样式 + HUD/弹窗
- `game.js`：游戏主逻辑（物理、碰撞、关卡、渲染）
